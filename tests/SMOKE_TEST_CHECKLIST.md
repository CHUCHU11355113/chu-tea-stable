# Chu Tea 项目冒烟测试清单 (Smoke Test Checklist)

**版本**: 1.0
**目的**: 在每次版本发布前，快速验证系统的核心功能是否正常工作，确保没有严重的功能阻断性 Bug。

---

## 测试说明

- **执行时机**: 本地开发合并到 `main` 分支后，或部署到测试/预发布环境后。
- **执行人员**: 开发人员或测试人员。
- **预期结果**: 所有测试用例均应通过。任何一项失败都意味着该版本存在严重问题，**禁止发布**，并立即返还开发修复。
